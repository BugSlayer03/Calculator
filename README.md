# 🧮 Calculator Web App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.
This project demonstrates basic frontend development concepts along with DOM manipulation and event handling.

## 🚀 Features

* Perform basic arithmetic operations
  (Addition, Subtraction, Multiplication, Division)
* Interactive button-based input
* Clear (`C`) button to reset input
* Dark / Light mode toggle 🌙☀️
* Responsive and clean UI design

## 🛠️ Tech Stack

* **HTML** – Structure of the application
* **CSS** – Styling and layout (including themes)
* **JavaScript** – Functionality and logic

## 📂 Project Structure

```
calculator/
│── index.html
│── style.css
│── script.js
│── sun-solid.png
│── moon-solid.png
```

## ⚙️ How It Works

* Button clicks are captured using event listeners
* Values are appended to the display dynamically
* `eval()` is used to calculate the final result
* Theme switching is handled using `classList.toggle()`

## 🖥️ How to Run

1. Clone the repository

```
git clone https://github.com/BugSlayer03/Calculator
```

2. Open the project folder

3. Run `index.html` in your browser

## ⚠️ Note

* This project uses `eval()` for calculation, which is fine for learning purposes but not recommended for production-level applications.
